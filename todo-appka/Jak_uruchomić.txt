1. pobrać Node.js https://nodejs.org



2. Zainstaluj Python jeśli nie masz:
https://python.org/downloads

Sprawdź:

python --version

Teraz:

Aktywuj środowisko:

Windows:

venv\Scripts\activate

Mac/Linux:

source venv/bin/activate

Zainstaluj FastAPI:

pip install fastapi uvicorn


Uruchom:

uvicorn main:app --reload



Struktura : 
project/
├── todo-app/   (React)
└── backend/    (FastAPI)


Ogólnie to jak jest gotowe to w 2 okienkach 2 różne cmd 
W 1 wchodze tam gdzie jest projekt ->..\projekt i najpierw npm install

potem wpisuje npm run dev
(tak powinno wyglądać)
